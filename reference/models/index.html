    Models Reference¶

    This section provides detailed documentation of the model architectures and configurations available in Tabular SSL.

    TabularSSL¶

    The main model class for self-supervised learning on tabular data.

    from tabular_ssl import TabularSSL

    Architecture¶

    The TabularSSL model consists of:

    1. Input Layer
    2. Handles mixed data types (numerical and categorical)
    3. Feature embedding layer

    4. Positional encoding

    5. Transformer Encoder

    6. Multi-head self-attention
    7. Feed-forward networks
    8. Layer normalization

    9. Residual connections

    10. Output Layer

    11. Feature reconstruction
    12. Task-specific heads

    Configuration¶

    The model uses Hydra's configuration system with the _target_ pattern for flexible component instantiation.

    Example Configuration¶

    _target_: tabular_ssl.models.TabularSSL

# Model components
event_encoder:
  _target_: tabular_ssl.models.encoders.EventEncoder
  input_dim: 10
  hidden_dims: [64, 32]
  output_dim: 16

sequence_encoder:
  _target_: tabular_ssl.models.encoders.TransformerEncoder
  input_dim: 16
  hidden_dim: 32
  num_layers: 2
  dropout: 0.1
  num_heads: 4

embedding:
  _target_: tabular_ssl.models.embeddings.FeatureEmbedding
  embedding_dims:
    - [5, 8]  # 5 categories, 8-dimensional embedding
    - [3, 4]  # 3 categories, 4-dimensional embedding

projection_head:
  _target_: tabular_ssl.models.heads.ProjectionHead
  input_dim: 32
  hidden_dim: 16
  output_dim: 8

prediction_head:
  _target_: tabular_ssl.models.heads.PredictionHead
  input_dim: 8
  num_classes: 2
  dropout: 0.1

    Methods¶

    train(data, batch_size=32, epochs=100, learning_rate=1e-4)¶

    Train the model using self-supervised learning.

    Parameters: - data (pd.DataFrame): Input data - batch_size (int): Batch size for training - epochs (int): Number of training epochs - learning_rate (float): Learning rate

    Returns: - dict: Training history

    predict(data)¶

    Make predictions on new data.

    Parameters: - data (pd.DataFrame): Input data

    Returns: - np.ndarray: Model predictions

    save(path)¶

    Save the model to disk.

    Parameters: - path (str): Path to save the model

    load(path)¶

    Load a model from disk.

    Parameters: - path (str): Path to the saved model

    Returns: - TabularSSL: Loaded model

    Model Components¶

    Feature Embedding¶

    from tabular_ssl.models.embeddings import FeatureEmbedding

embedding = FeatureEmbedding(
    input_dim=10,
    embedding_dim=64
)

    Transformer Encoder¶

    from tabular_ssl.models.encoders import TransformerEncoder

encoder = TransformerEncoder(
    input_dim=64,
    hidden_dim=256,
    num_layers=4,
    num_heads=4
)

    Task Heads¶

    from tabular_ssl.models.heads import TaskHead

head = TaskHead(
    input_dim=256,
    output_dim=10
)
